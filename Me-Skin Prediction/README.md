## Installation
**1. Install using fastAPI**
```Terminal cmd
pip install fastapi
```

2. Install using requirements.txt
```Terminal cmd
pip install -r requirements.txt
```

**How to run by default**
```Terminal cmd
python -m uvicron app:app --reload
```
